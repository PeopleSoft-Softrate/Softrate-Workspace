Loss of Pay (LOP) Rules

Deducted for unauthorized leaves — pending approvals and absences without any leave submission.

Company public holidays and weekly off days (from Holiday Calendar) are automatically excluded from LOP calculation.

Employee LOP

Apply LOP deduction for employees with unauthorized leave.

%

100% = deduct full day's salary per unauthorized day.

₹

Fixed amount deducted per unauthorized leave day.

Used for formula purposes only to calculate per-day salary (Basic ÷ Working Days).

Example: Basic ₹35,000 → Per day = ₹{{ (35000 / (workingDaysEmployee() || 26)) | number:'1.0-0' }} → LOP/day = ₹{{ (35000 / (workingDaysEmployee() || 26)) * ((lopPercentageEmployee() || 0) / 100) | number:'1.0-0' }}
Example: 3 unauthorized days → LOP = ₹{{ (lopAmountEmployee() || 0) * 3 | number:'1.0-0' }}

Intern LOP

Apply LOP deduction for interns with unauthorized leave.

%

100% = deduct full day's stipend per unauthorized day.

₹

Fixed amount deducted per unauthorized leave day.

Used for formula purposes only to calculate per-day stipend (Basic ÷ Working Days).

Example: Basic ₹12,000 → Per day = ₹{{ (12000 / (workingDaysIntern() || 26)) | number:'1.0-0' }} → LOP/day = ₹{{ (12000 / (workingDaysIntern() || 26)) * ((lopPercentageIntern() || 0) / 100) | number:'1.0-0' }}
Example: 3 unauthorized days → LOP = ₹{{ (lopAmountIntern() || 0) * 3 | number:'1.0-0' }}
